Solar Forcasting Data Science ML Project | Python

May, 2022

This was a group project I did during my Fourth Semester as part of the Data science course.Github link.

In part 1, we learned how to find data, mine using APIs and convert it into a more accessible format. We then went ahead and cleaned the data and performed statistical analysis and data visualisation of the same.

In part 2, we learned about various machine learning models and their pros and cons. We applied models such as Multiple Linear Regression, SVR, XGBoosted Decision Tree and Random Forest. We then compared the results and performance of all the models.

A further improvement can be done by optimizing hyperparameters of the various models.
